# zh-cn

## 目录说明
本目录存放对应数字发行的简体中文字幕，按分辨率与特性组织，便于检索与自动化处理。文件必须遵循[仓库命名规范](/docs/REPO_SUBTITLE_NAMING_CONVENTION.md)并使用 UTF-8 编码。

## 重要说明
- 本目录下所有以 `eng-chs` 和 `chs` 结尾且来源于 `chs-eng` 的文件均为由脚本自动生成的衍生版本，请勿直接编辑这些衍生文件。若需修正字幕内容或时间轴，请编辑对应的 `chs-eng` 源文件，保存后脚本会自动生成 `eng-chs` / `chs` 衍生文件。  
- 字幕制作者上传文件时应保持与仓库中既有文件一致的换行符格式（不要在同一仓库中混用 LF/CRLF），确保换行符格式统一以避免自动化处理与版本比较出现差异。  
- `.ass` 文件为特效字幕。当 `.ass` 与 `.srt` 同时发布且初始内容同步时，今后对字幕的关键更改必须同时同步到这两个文件（例如：若编辑 `1080p.chs-eng.ass`，应同时手动更新对应的 `1080p.chs-eng.srt`，反之亦然），以保证无特效与有特效版本内容一致。

## 快捷帮助

- **上传字幕**  
  若你有权限编辑此目录，点击右上角的 **Add file** 上传字幕文件。上传时在 **Commit message** 中写明本次提交的具体变更，便于日后检索。

- **下载字幕**  
  点击文件名（例如 `1080p.chs.srt`），在文件页面右上角点击下载图标即可保存。  
  > 若需详细的 GitHub 使用说明，请参阅组织内的 “GitHub 使用帮助” 百科。

- **贡献字幕**  
  若你并非字幕组成员但想贡献字幕或提交更改，请参阅组织内的 **“如何贡献”** 百科。
